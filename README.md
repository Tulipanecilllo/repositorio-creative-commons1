# Taller de Licencias Creative Commons

Autor: Oleidy Jimenez M, Lia Nicole, Francis Mendez , Samuel Sanchez, Ronal Mojica
Tema: Ética, Ciberseguridad y Derechos de Autor en la Web  

## Descripción
Este repositorio fue creado como parte del Taller de Licencias Creative Commons. 
El objetivo es identificar diferentes tipos de licencias y explicar las condiciones de uso de imágenes obtenidas de la web.
